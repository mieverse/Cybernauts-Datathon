## 📌 Project Overview
This repository contains the end-to-end machine learning solution developed by team **Syntax_Error** to predict customer churn for FictiPay during the April 2024 validation window. 

Digital wallets operate in high-frequency, highly volatile user environments. By restructuring multi-gigabyte KYC registers, daily balance histories, and raw transactional logs using parallel computing frameworks, we extracted high-fidelity behavioral features. Our final predictive engine leverages a strict validation strategy and a blended meta-ensemble to ensure stable, top-tier performance on the hidden leaderboard.
